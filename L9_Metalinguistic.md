# Layer 9: Metalinguistic Layer

**Purpose:**  
Glyphs represent symbols about symbols — self-describing logic, syntax morphisms, and symbolic recursion.

**Core Concepts:**
- Symbol-to-symbol mapping
- Syntax reflection
- Self-encoding systems
- Language of language

**Formal Representation:**
- Nodes = metasymbols
- Field = grammar transformer
- Enables meta-code, meta-poetry, syntax flow
